# PyCine - Frontend

Interface de usuário desenvolvida em **Svelte/SvelteKit** para interagir com a API do PyCine.

> **Nota:** Este módulo depende do Backend (Python/FastAPI) estar rodando para exibir os dados corretamente.

## Pré-requisitos

- Node.js (v16 ou superior)
- NPM

## Como executar

Abra o terminal nesta pasta e execute os comandos abaixo:

### 1. Instalar dependências
Baixa as bibliotecas necessárias para o projeto rodar.
```bash
npm install

### 2. Iniciar a aplicação

```sh
npm run dev
```

iniciar e abrir no browser
```sh
npm run dev -- --open
```
